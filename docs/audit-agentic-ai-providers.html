Limitation critique : tools + streaming

La doc DashScope indique que "tools parameter cannot be used with stream=True simultaneously". Cela signifie que le mode agentic de ClawBot (qui stream les tokens pendant le tool loop) ne fonctionnerait pas directement. Solutions possibles :

  • Mode non-streaming pour les requetes avec tools (acceptable si latence premiere reponse < 5s)
  • Verifier si cette limitation a ete levee dans les modeles Qwen3/3.5 recents (doc potentiellement obsolete)
  • Utiliser le DashScope native API au lieu du mode OpenAI-compatible (plus de features)

Agentic coding : Heritage Devstral — entraine specifiquement pour les taches de code agentic (tool loops, file manipulation, debugging).

Performance : 40% reduction de latence end-to-end et 3x le throughput vs Small 3 (MoE = seulement 6B actifs malgre 119B totaux).

Verdict Mistral Small 4 pour ClawBot

Candidat serieux comme provider mid-range europeen. Mistral Small 4 combine reasoning, vision et coding dans un seul modele a $0.15/$0.60 — moins cher que Claude Haiku ($0.80/$4.00) avec des capacites proches. Le reasoning_effort configurable permet d'adapter la qualite au contexte — rapide pour le chat, profond pour les taches complexes.

10.2 — Routing par Plan recommande

Constat business : Claude est sur-dimensionne pour 99% des usages ClawBot

La cible ClawBot n'est pas un developpeur ou un power user — c'est un chef de TPE, un artisan, un independant, une famille qui achete un boitier a 30€. Ses requetes quotidiennes : "envoie un message a Pierre", "quel temps demain", "resume ce mail", "mets un timer". Pour ces taches triviales, Qwen Flash, Mistral Small 4 ou Kimi sont indiscernables de Claude. L'utilisateur ne percevra jamais la difference.

Consequence : Claude ($0.80-$75/M tokens) en provider principal sur tous les plans payants est du sur-engineering technique sans benefice percu par l'utilisateur final. La marge s'effondre pour rien. Claude doit devenir un add-on premium ou un fallback intelligent pour les rares cas complexes, pas le defaut.

11Verdict & Recommandations

Changement strategique majeur : Claude n'est plus le provider par defaut

L'utilisateur cible ClawBot — chef de TPE, artisan, independant, famille — effectue des taches simples a 99%. "Envoie un message", "quel temps demain", "resume ce mail". Pour ces usages, Qwen, Mistral ou Kimi sont indiscernables de Claude. Le cout Claude ($0.80-$75/M) n'est justifie que pour les rares cas complexes (agents multi-tool, coding, MCP). Claude devient un add-on premium, pas le defaut.

Recommandation #1 — Mistral Small 4 comme provider principal (tous plans Perso)

Impact : A $0.15/$0.60, Mistral Small 4 couvre 99% des taches avec reasoning configurable (reasoning_effort=none pour le chat, high pour les reflexions). Heberge en Europe (RGPD), excellent en francais, Apache 2.0. Cout mensuel estime par utilisateur actif : ~$1-2/mois vs $27/mois avec Claude Sonnet. La marge passe de negative a confortable sur tous les plans Perso.

Action : Ajouter mistral dans PROVIDER_URLS + PROVIDER_KEYS. Router Perso 7.99 / 30 / 80 vers mistral-small-latest. Gerer reasoning_effort dans llm_proxy.

Recommandation #2 — Qwen Flash/Plus comme fallback budget et plan Free

Impact : Qwen Flash a $0.05/$0.40 est imbattable sur le Free tier. Qwen Plus a $0.40/$1.20 offre un excellent fallback si Mistral est indisponible. La compatibilite OpenAI rend le failover transparent.

Risque : La limitation tools + stream necessite verification sur Qwen3.5 recents.

Status : DEPLOYE Qwen deja dans PROVIDER_URLS + PLAN_ROUTING.

Recommandation #3 — DeepSeek V3.2 comme fallback reasoning

Impact : Le "Thinking with Tools" unique de V3.2 en fait le meilleur fallback pour les taches complexes. Deja configure dans PROVIDER_URLS.

Status : DEPLOYE dans PLAN_ROUTING pour les plans Pro.

Recommandation #4 — Claude reserve aux plans Pro 189€+

Impact : Claude reste le meilleur en fiabilite tool calling (9.5/10) et suivi d'instructions complexes. Mais ce niveau de qualite n'est perceptible que sur des taches avancees : agents coding multi-rounds, MCP, SSH, credential vault. Seuls les plans Pro a 189€+ justifient le cout API et ont des utilisateurs qui percevront la difference.

Consequence : Sur les plans Perso, Claude devient un fallback silencieux (si Mistral echoue) ou une option activable explicitement — jamais le provider par defaut.

Recommandation #5 — Kimi conserve pour le web search gratuit

Impact : Kimi reste le seul provider offrant un web search natif gratuit ($web_search). Utile comme outil complementaire mais surclasse par Qwen/Mistral sur tous les autres criteres agentic.

Status : DEPLOYE — fallback + web search en production.
